Java for C++ programmers: Getting started with Java

Variables & Data Types

Variables and Data Types in Java are very similar to their C++ counterparts.

Declaring a variable, assigning to it, or accessing it is the same as that in C++.

Java has the following data types that are present in C++ as well:

  • int
  • float
  • long
  • double
  • char

The above data types are known as primitive data types. There is something known as reference data types as well. We will learn more about it in a future chapter.

Note:

  • Unlike C++, there is no concept of unsigned in Java.
  • Boolean:
    • There is a data type in Java to represent boolean values.
    • The data type is boolean and its value can be either true or false.
    • 1 or 0 are not valid boolean values in Java and cannot be used to represent true/false.
    • boolean is also a primitive data type.
  • Integer numbers are treated as int by default.
    Example:
    long longVariable = 13800000000;
    Before being assigned to longVariable, the number is an int value. Hence, the above code would fail as the number is too big for int data type and int to long typecasting is not done automatically in Java.
    We have to instead write it as 13800000000L to have it interpreted as a long number.
    Example:
    long longVariable = 13800000000L;
  • Decimal numbers are treated as double by default.
    Example:
    float floatVariable = 3.141592;
    double to float typecasting is not done automatically in Java. Hence, the above code would fail.
    We have to instead write it as 3.141592f to have it interpreted as a float number.
    float floatVariable = 3.141592f;

Java has two types of typecasting:

  • Widening Casting
    • A smaller data type being typecasted to a larger data type.
      Examples:
      • int to long
      • float to double
      • int to float
      • char to int
    • Order: char -> int -> long -> float -> double
  • Narrowing Casting
    • A larger data type being typecasted to a smaller data type.
      Examples:
      • long to int
      • double to float
      • float to int
      • int to int
    • Order: double -> float -> long -> int -> char

In C++, both widening and narrowing casting happens automatically.

Example
int num = 42;
double doubleNum = num; //Widening Casting without explicit typecasting
int num1 = doubleNum; //Narrowing Casting without explicit typecasting

double doubleNum1 = (double) num; //Widening Casting with explicit typecasting
int num2 = (int) doubleNum1; //Narrowing Casting with explicit typecasting

Typecasting can be done in C++ with or without explicitly mentioning the data type. In Java, narrowing casting necessarily requires manual typecasting. The above code will throw an error in Java. We have to manually typecast from double to int in Java.

Example
int num = 42;
double doubleNum = num; //Widening Casting without explicit typecasting
int num1 = (int) doubleNum; //Narrowing Casting with explicit typecasting

double doubleNum1 = (double) num; //Widening Casting with explicit typecasting
int num2 = (int) doubleNum1; //Narrowing Casting with explicit typecasting
